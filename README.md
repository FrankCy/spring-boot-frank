# Spring Boot Frank #

## 前言 ##
Spring Boot 是简化Spring应用的创建、运行、调试、部署等一系列问题而诞生的产物，```自动装配的特性可以让开发更专注于业务本身，而不是外部的XML配置，遵循规范开发，引入相关的依赖就可以轻易的搭建出一个WEB工程。```
