Redis常用功能

## 功能 ##
地址|说明
:---|:--
http://localhost:8077/redis/index?flag=update| 单节点修改
http://localhost:8077/redis/index?flag=delete| 单节点删除
http://localhost:8077/redis/index?flag=select| 单节点查询
http://localhost:8077/redis/index?flag=insert| 单节点新增
